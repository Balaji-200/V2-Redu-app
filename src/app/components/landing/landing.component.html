V2 Redu

Learn to identify and prioritize patient care during mass causality incidents and disasters. This interactive video simulation software helps to engage and learn in an interesting way while trying to improve patient care outcomes.

About

The following program will take you to video simulation classes, quizzes, case scenario based interactive videos. There is time limit for certain case scenarios once you become expert to the simulation classes which would help you to improve your skills for identifying and tagging in triage system. You can attempt the program more than once; however, you must complete all steps of the program before you can repeat any of the previous steps.
